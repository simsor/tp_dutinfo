Fonction puissance_recur(a : Réel, n : Entier) Retourne Réel
Début
    Avec b : Réel
    
    Si n  = 0 Alors Retourne 1
    Sinon
		Si n = 1 Alors Retourne a
		Sinon
			b <- puissance(a, n/2)
			Si n mod 2 = 0 Alors
				Retourne b * b
			Sinon
				Retourne a * b * b
			Fin Si
		Fin Si
    Fin Si
Fin puissance_recur
